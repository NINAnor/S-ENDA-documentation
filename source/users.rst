=====================
Users 
=====================

There are 2 types of users:
 1. Those that create / produce data
 2. Those that consume data 

Producers:
* Volvo car data
* Private weather station data (Netatmo)
* Vegvesen
* Weather model data
* Ocean model data
* Warnings forecast 


Consumers:
* Climate Predictions (klima i norge 2100)
* Klimavakt (MET)
* Developer
